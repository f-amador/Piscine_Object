==**Modules**==

The following modules must be validated in this order: 

- [[Module00]] - [[Encapsulation]] - Encapsulation This module will introduce you to the concept of encapsulation.

-  [[Module01]] - Relationship This module will introduce you to how variables/attributes can be linked together, using specific kind of relationship.

- Module 02 - UML This module will introduce you to a way to represent graphicaly software architecture. 

- Module 03 - SOLID This document will introduce you to the five SOLID principles. 

- Module 04 - Design Pattern This module will introduce you to the Gang of Four design patterns.

- Module 05 - Pratical work This module is an exercice to use what you will have learned in the previous 5 modules
